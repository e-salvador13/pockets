# Pockets — AI Personal Finance

YNAB's philosophy meets AI execution. Budget less, understand more.

## What's Different
- **Conversational onboarding** — 5 AI turns, not 20 checkbox screens
- **AI smart budgets** — analyzes your history, proposes a plan
- **Auto-rebalancer** — overspent somewhere? AI adjusts, you approve
- **Proactive coaching** — app reaches out to you, not the other way around
- **Psychology-first** — understands your money archetype, adapts to you

## Stack
- Next.js 15 (App Router)
- Claude (Anthropic SDK)
- Zustand (state)
- Tailwind + shadcn/ui
- Plaid (bank connection — planned)

## Getting Started
```bash
npm install
npm run dev
```
